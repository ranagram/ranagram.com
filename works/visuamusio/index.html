VISUAMUSIO API

VISUAMUSIO is an iPad application for the creation of visual music animations. We were in charge of the development of the generative audio system and programming both the database server API and the sharing website.We developed the online database system for allowing people to share their visual music compositions in collaboration with WOW Inc.

WOWよりリリースされた、音楽とアニメーションをテーマにしたiPadアプリ「VISUAMUSIO」のシェアリングサイト。ラナグラムではサーバサイドでの音響合成システム、データベースAPI及びシェアリングサイトの構築を行いました。ユーザーからの送信データより生成した楽曲をオンラインで共有できます。
