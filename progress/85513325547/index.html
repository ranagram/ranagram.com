An experimental work that attempts to control the 3D graphics with hand gesture using Leap Motion. The custom software is built with openFrameworks.

サンプラスデザインとの共同イベント「GRAMARAS 2013」のための実験作品。小型のモーションセンサー Leap Motionを使い、手のジェスチャーを元にヴィジュアルをリアルタイムコントロールするプログラムを作りました。

Design & Code: Shinobu Tanno
